# devcard

Developer identity cards for [Claude Code](https://claude.ai/code).

A plugin that turns your GitHub profile into an interactive developer card - with an AI-written review of your work, a Claude Code collaboration fingerprint, and an activity heatmap.

[Browse the gallery](https://devcard-community.github.io/registry/) &middot; [Get the plugin](https://github.com/devcard-community/devcard)
